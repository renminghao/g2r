Section1.2

seller-tree

示例

对数据结构的要求，只要是树型的即可，可以通过 labelKey、valueKey、childrenKey 控制字段映射

var treeData = [
  {
    name: '节点1',          //可以通过labelKey属性，通知组件用数据中的name属性渲染节点的文字部分
    id: 1,                  //可以通过valueKey属性，通知组件用数据中的id属性作为节点的value部分
    children: [{            //可以通过childrenKey属性，通知组件从数据中的children属性查找节点的孩子节点
      name: '节点3'
      id: 3
    }]
  },
  {
    name: '节点2',
    id: 2,
    children: [{
      name: '节点3',
      id: 3
    }]
  }
]

单选、只有叶子节点都能被选中，试用于叶子节点的数据筛选，例如根据叶子类目创建商品

<Tree multiple={false} checkedType="leaf" showCheckbox={false} labelKey="name" valueKey="id" dataSource={treeData} />

单选、只有叶子节点都能被选中，试用于叶子节点的数据筛选，例如根据叶子类目创建商品,子节点为模板渲染

<Tree multiple={false} checkedType="leaf" showCheckbox={false} labelKey="name" valueKey="id" dataSource={treeData} labelKeyTemplate={$getTemplate('render') />

<Template name="render">{id}---{name}</Template>

单选、所有节点都能被选中，试用于树形结构的数据管理，例如对树种任意节点的筛选、编辑操作

<Tree multiple={false} checkedType="all" showCheckbox={false} labelKey="name" valueKey="id" dataSource={treeData} />

多选、只有叶子节点能被选中、显示checkbox，试用于于从树形数据中筛选出多条数据，例如从树型的功能点列表中筛选出需要的功能点

<Tree multiple={true} checkedType="leaf" showCheckbox={true} labelKey="name" valueKey="id" dataSource={treeData} />

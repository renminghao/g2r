Table

参数 说明 类型 可选值 默认值
className 表格class string
dataSource 数据数组 Array
columns 表格列的配置描述，具体项见下表 Array 无
rowSelectable 是否可以选择行 Boolean false
onSelectAll 表格全选后的回调, 参数分别为是否选中，表格中选中的数据 Function(selected, selectedRows){} 无
onSelect 表格选中一行后的回调，参数分别为改行记录，是否选中，表格中选中的数据 Function(record, checked, selectedRows){} 无
rowSelectedWhen 默认选中的行的条件 Function (record)=>{return record.id===1} 无
rowSelectDisabledWhen 禁止选中的行的条件 Function (record)=>{return record.id===1} 无
styleCfg 样式控制的一些参数，目前只支持传入width(type:number) Object 无
hideTableHead 是否隐藏表头，设置为true隐藏表头 Boolean false
fixWidth 是否固定宽度(设置该参数后如果表格每列宽度加起来大于设置的表格宽度，那么表格自动水平滚动，如果小于，每一列按比例缩放到设置的表格宽度。设置后每一列的宽度必须设置)
onSort 表格点击排序后的回调，参数如下：{field: 'a', column: [object], order: 'descend'},其中field表示排序的字段，column为排序的这一列的所有属性，order为排序方式，descend为降序，ascend为升序 Function 无
noapTemplate 数据为空时展示的内容 `function string` 暂无数据
sortOnLocal 数据是否本地排序 Boolean false
sortSignLocation 排序按钮位置 string left或right left
columnLine 是否有列分割线，设为true时出现列分割线 Boolean false

Table-多级嵌套

除了上述参数，多级嵌套Table还有以下可用参数，用于控制嵌套展开交互逻辑

参数 说明 类型 默认值
onExpandedRowsChange(record) 行展开状态变化回调 Function noop
expandedRowKeys 设置行展开keys Array
onExpandButtonHover(record) 行展开的+号按钮hover事件回调 Function noop
onRowhover(record) 表格行hover事件回调 Function noop
onRowClick(record) 表格行click事件回调 Function noop
Record对象

除了默认行数据之外，还包含以下参数

参数 说明 类型 默认值
key 包括展开行在内递增字符串 String 比如row1,row2
level 数据的层级 Number 第一层级为1，比如非嵌套Tabel，所有数据level都是1
expandable 是否可展开，有children并且有item的数据为可展开数据 Boolean

Column

参数 说明 类型 可选值 默认值
title 列头显示文字 String or React.Element
dataIndex 列数据在数据项中对应的 key String
render 生成复杂数据的渲染函数，参数分别为当前列数据，列索引，@return里面可以设置表格行/列合并 Function(record, index) {}
colSpan 表头列合并,设置为 0 时，不渲染 Number
width 列宽度 String or Number 无
className 列的 className String 无
textWrap 文字过多时不换行，自动隐藏超出的内容（使用这个参数时表格需要设置fixWidth，同时这一列的宽度必填） Boolean false
sortable 是否可以排序 Boolean false

使用

RTemplate中使用

<Table dataSource={data}
      rowSelectable="{true}"
      rowSelectedWhen="{(record)=>{return record.id===1}}"
      rowSelectDisabledWhen="{(record) => {return record.id===3}}"
      onSelect={(...args)=>{console.log(...args)}}
      onSelectAll={(...args)=>{console.log(...args)}}
      onSort="{v => console.log(v)}">
      <Th title="表头1" dataIndex="a" sortable="{true}" width="100" />
      <Th title="表头1" dataIndex="a" width="100"/>
      <Th title="表头2" dataIndex="b" width="50"/>
      <Th title="表头3" dataIndex="c" width="50"/>

详细说明

行选择

注意：在数据中加入{selectable:false}来禁止该行选中的做法已不再推荐，要禁止某行选中，请使用rowSelectDisabledWhen参数

  let columns = [
      {"title": "表头1", "dataIndex": "a"},
      {"id": "123", "title": "表头2","dataIndex": "b", "dataType": "num","render": ""},
      {"title": "表头3", "dataIndex": "c"},
      {"title": "操作", "dataIndex": "d", "render": function() {
          return <a href="#">操作</a>
      }}
    ];

    let data=[
      {"a": "123"},
      {"a": "cdd","b": "edd"},
      {"a": "1333","b": "ad","c": "eee","e": 2}
    ];

    ReactDOM.render(
        <Table rowSelection={true}
          columns={columns}
          dataSource={data}
          rowSelectDisabledWhen={(record)=>{return record.a ==="cdd"}}></Table>,
        container
    )

单元格跨行/跨列

如果需要某个单元格跨行/跨列，那么需要在column数据的render方法中指定colSpan或rowSpan

  let columns = [
      {"title": "表头1", "dataIndex": "a", "sortable": true, "width": 200, "className": "sui-table-align-right", "render": function(record, index) {
        if (index === 0) {
          return <span colSpan="2">{record.a}</span>
        } else {
          return <span>{record.a}</span>
        }
      }},
      {"id": "123", "title": "表头2","dataIndex": "b", "dataType": "num","render": "", "width": 100},
      {"title": "表头3", "dataIndex": "c", "width": 200},
      {"title": "操作", "dataIndex": "d", "render": function(record, index) {
          if (index === 1) {
            return <a rowSpan="2" href="#">操作</a>
          } else if (index === 2) {
            return <a rowSpan="0"></a>
          } else {
            return <a href="#">操作</a>
          }

      }, "width": 100}
    ];

    let data= [
    {"a": "123", "b": "egg","c": "cdd"},
    {"a": "cdd","b": "edd"},
    {"a": "1333","b": "ad","c": "eee","e": 2}
  ];

    ReactDOM.render(
        <Table rowSelection={true} columns={columns} dataSource={data}></Table>,
        container
    )

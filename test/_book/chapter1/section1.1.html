0. 注意事项

  1. 普通的Input、Select、DateTimePicker已在Form内部集成，如果您在RTemplate中使用Form那么直接使用就可以；如果在JS中使用需要require Form组件内部的版本，否则下面的布局、校验均无法生效。表单内置组件详情请点击这里。
  2. 所有的表单内置组件都可以使用表单字段属性，详细的使用说明请点击这里。

2 布局说明

表单默认使用flex+rem布局。

  1. 如果在Form标签内部直接写表单元素，那么表单就是垂直单列布局；
  2. 单列布局时表单元素宽度固定，可以通过size属性调整宽度；
    • size的默认值为normal，可选smaller,small,large；
    • size也可以自定义数字，表单元素的宽度就是自定义的大小；
    • size属性不会对textarea生效
  3. 使用View标签嵌套可以实现水平多列布局，View标签内部的元素水平排列，且宽度自适应；
  4. 多列布局的时候，可以通过col属性调整每一列表单元素的宽度。
    • 每一项表单元素前面的label默认占两列宽度，如果宽度不够出现文字换行的情况，可以通过labelCol这个属性设置label的宽度。比如labelCol=3，那么label的宽度就占三列了。

<Form ref="form"  
      action="//dip.alibaba-inc.com/api/v2/services/schema/mock/17544"
      onSuccess="{(resp) => {console.log(resp)}}"
      onChange="{v => console.log(v)}"
      method="post">

    <Input type="text" name="small" size="small" label="size=small" required="true"  placeholder="170px"/>
    <Input type="text" name="normal" size="normal" label="size=normal" required="true"  placeholder="240px"/>
    <Input type="text" name="large" size="large" label="size=large" required="true"  placeholder="470px"/>

    <View>
      <Input name="email" format="email" label="邮箱：" required="true" placeholder="均分宽度"/>
      <Input name="tel" format="tel" label="电话：" required="true" placeholder="均分宽度" />
      <Input name="number" format="number" label="数字：" required="true" placeholder="均分宽度" />
      <Input name="url" format="url" label="url：" required="true" placeholder="均分宽度" />

  1. 所有的内置组件都可以使用这些属性；

4. 属性说明

4.1 表单属性

名称 描述 类型 是否必须 默认值
action 数据提交地址 String × ''
method 数据提交方法 get│post│form × 'get'
classPrefix 类前缀，默认 "sui-" String × 'sui-'
className 自定义类名, 默认 "" String ×
confirm 提交前的确认信息，用户点击确定过后才会提交。false代表不提示 String│Boolean × false
autoTrim 自动移除表单提交的字段前后的空格 Boolean × false
onSubmit 表单提交前的回调，可以return false 禁止表单提交 Function × () => { return true; }
onLoad 数据加载完成后的回调函数 Function × () => {}
onSuccess 数据加载成功（res.success === true），的回调 Function × () => {}
onReset 表单重置后的回调 Function × () => {}
onFail 数据加载失败的回调函数，可从 res.msg 读取错误消息 Function × data => { Modal.alert(data && data.msg ? data.msg : '系统错误'); }

arraySubmitType 简单数组的提交格式，比如待提交数据是 { arr: [1, 2] } json：数据提交格式为 arr=[1,2] string: 数据提交格式为 arr=1,2, 分隔符通过 arraySubmitType 控制 form: 数据提交格式为 arr=1&arr=2 formArray: 数据提交格式为 arr[]=1&arr[]=2 json│string│form│formArray × 'form'

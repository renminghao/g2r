PropTypes

名称 描述 类型 是否必须 默认值
active bool,是否为按住状态样式 Boolean ×
allowDisabledClick 是否允许在Disabled状态下Click事件被触发(该配置仅在非button标签的时候才有效,button标签会被默认阻止click事件) Boolean × false
className 添加的类名 String × ''
disabled true 为 disabled Boolean ×
href 有则显示为a标签,并且href等于此值 String ×

mode input a button String ×
noSuibtn 不显示sui-btn类 Boolean ×
onClick 事件 Function ×
size 默认为空 small│large│normal ×
skin 默认为空 primary│link-primary│link│icon│normal ×
value 如果有value则children读取value,无则读取children本身 String ×

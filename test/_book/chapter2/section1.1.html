Section1.1

seller-transfer

配置

名称 描述 类型 是否必须 默认值
title 标题 String 否 穿梭框
buttonText 操作按钮文案 Array 否 [确定，取消]
successEvent 第一个操作按钮点击回调 Function 否 null
cancelEvent 第二个操作按钮点击回调 Function 否 null
value 被选中的节点value，数组，或者逗号分隔的value列表 String│Array│Number × []
dataSource 树的数据 Array × []
onExpand 节点被展开时的回掉，第一个参数为节点的数据对象 Function × blank
onCollapse 节点被收起时的回调，第一个参数为节点的数据对象 Function × blank
onChange 选中状态发生变化时的回调，第一个参数为节点的value，第二个参数为节点的数据对象 Function × blank
checkedType 选中类型，all：所有节点可被选中，leaf：只有叶子节点可被选中 all│leaf × 'leaf'
multiple 是否多选 Boolean × false
showCheckbox 可以被选中的节点，是否显示checkbox Boolean × true
readOnly 只读，不能勾选 Boolean × false
defaultExpandedType 初始展开状态，checked，被选中的分支会展开，all，全部展开，none，全部不展开 all│none│checked × 'checked'
childrenKey 设置子节点的字段名 String × 'children'
labelKey 设置用于显示的文本字段 String × 'label'
valueKey 设置用于提交的数据字段 String × 'value'
searchKey 设置搜索匹配的节点 String × 'any'
delimiter value作为字符串传入是，多个value的分隔符，默认英文逗号 String × ','
className 组件最外层节点的自定义类名 String × ''
checkedClassName 被点击选中节点的自定义类名 String × ''
matchedClassName 搜索命中的节点的自定义类名 String × ''
nodeClassName 所有节点的自定义类名 String × ''
leafNodeClassName 叶子节点的自定义类名 String × ''
searchValue 用于搜索匹配的输入内容 String ×
nodeRender 节点渲染模板 Function × null

基本用法

      <Transfer 
      dataSource={data1} 
      checkedType={'all'} 
      title={"这是一个测试"} 
      labelKey={"name"} 
      valueKey={"id"} 
      multiple={true}
      value={[1,2,3,4,5]}
      onChange={(...args)=>{console.log(...args)}} 
      successEvent={(target)=>console.log(target)} 
      cancelEvent={(resultItem)=>console.log(resultItem)} 
    />

将需要展现操作的数据作为属性传入

  {
        data : [{
          name : '夹克标题',
          select : 1,
          showList : true,
          data : [{
            name : '夹克1',
            select : 0,
            data : [{
              name : '夹克13',
              select : 0,
            },{
              name : '夹克41',
              select : 1
            }]
          },{
            name : '夹克2',
            select : 1,
          }]
        }],
        name : '这是个测试哦'
      }
  }

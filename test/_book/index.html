单元格跨行/跨列

  let columns = [
      {"title": "表头1", "dataIndex": "a", "sortable": true, "width": 200, "className": "sui-table-align-right", "render": function(record, index) {

          return <span colSpan="2">{record.a}</span>

          return <span>{record.a}</span>
        }

2 布局说明

表单默认使用flex+rem布局。

  1. 如果在Form标签内部直接写表单元素，那么表单就是垂直单列布局；
  2. 单列布局时表单元素宽度固定，可以通过size属性调整宽度；
    • size的默认值为normal，可选smaller,small,large；
    • size也可以自定义数字，表单元素的宽度就是自定义的大小；
    • size属性不会对textarea生效
  3. 使用View标签嵌套可以实现水平多列布局，View标签内部的元素水平排列，且宽度自适应；
  4. 多列布局的时候，可以通过col属性调整每一列表单元素的宽度。
    • 每一项表单元素前面的label默认占两列宽度，如果宽度不够出现文字换行的情况，可以通过labelCol这个属性设置label的宽度。比如labelCol=3，那么label的宽度就占三列了。

https://github.com/JedWatson/react-select 的改造版

主要功能

  • 可以代替 html 的 select
  • 支持多选
  • 支持模糊查询，本地或者接口数据都可以
  • 支持按拼音匹配
  • 支持创建自定义的option

注意

  • 这个组件因为功能完善，所以代码体积较大，如果只是需要一个简单的代替 html 的 select 的 react 组件，可以使用 mui/seller-select 。

示例

基本用法同 html select

    <Option value="1">选项1</Option>

将options作为属性传入

[{
   value: 1,
   label: '标题'
}]

<Select name="searchInput11"

支持多选

        name="searchInput11"
        options={options}
        defaultValue="1"
        valueKey="value"
